# 🛒 My Shopping List App – Kotlin & Jetpack Compose

A beautifully crafted, modern **Shopping List app** built using **Kotlin** and **Jetpack Compose**. This app allows users to add, edit, and delete shopping items with a clean, responsive UI and real-time updates — perfect for learning modern Android development.

---

## ✨ Features

- ✅ Add shopping items with name and quantity  
- ✅ Edit items directly in the list  
- ✅ Delete items with a single click  
- ✅ Clean and responsive UI using **Jetpack Compose**  
- ✅ **Material 3 Design** with elegant Card views and dialogs  
- ✅ Reactive state handling with `remember` and `mutableStateOf`  

## 🧑‍💻 Tech Stack

- **Kotlin** – Android's official programming language  
- **Jetpack Compose** – Modern declarative UI toolkit  
- **Material 3** – UI Components and Design system  
- **State Management** – `remember` + `mutableStateOf`  
- **Scaffold**, **LazyColumn**, **Dialogs**, **Cards**, **OutlinedTextField**
